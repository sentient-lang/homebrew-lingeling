# Homebrew Lingeling

[Homebrew](http://brew.sh/) formulae to install the
[Lingeling, Plingeling and Treengeling](http://fmv.jku.at/lingeling/) SAT solvers.

```console
$ brew tap sentient-lang/lingeling
$ brew install lingeling
```
